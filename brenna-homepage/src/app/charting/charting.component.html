Examples of Observable Plot with Angular!


Select a feature to compare against the different species:

This section highlights how easy it is to adjust which features you are comparing in order to quickly identify which features are the best classifiers for a given dataset! For this data, you can visualize
each feature for each penguin species to see what is the best way to distinuguish the three species, Chinstrap, Adelie, and Gentoo from one another!

Select a chart to visualize the Olympic Data:

This scatter plot highlights the 'tip' feature that allows you to make the charts interactive. In this plot, the extra channels 'name' and 'sport' are added to the tooltip when the mouse hovers over each point.

This bar chart highlights the benefit of the inline transformations Observable Plot provides by grouping on sport and classifying by sex. The code for this chart also heavily relies on the inline CSS styling
of Observable Plot in order to produce a more visually pleasing and organized plot. For example, styling is used to rotate the x-axis labels by 90 degrees, change the font size of the x-axis labels, and
add padding inbetween each sport grouping.

This chart highlights the inline data transformation feature of Observable Plot by using the grouping feature and displaying a chart that groups traffic data both on location and hour in order to create a
heatmap that depicts the busiest times of the day per location.
